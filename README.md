## 🛠️ Tech Stack

### Frontend

- **Framework:** React (Vite)
- **Code Editor:** `react-simple-code-editor` & `prismjs`
- **Markdown Rendering:** `react-markdown` & `rehype-highlight`
- **HTTP Client:** Axios

### Backend

- **Environment:** Node.js
- **Framework:** Express.js
- **AI Integration:** Google Gemini API (or your respective AI model)
